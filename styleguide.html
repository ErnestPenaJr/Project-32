Overview

The DoCM Room Reservation System design system is built on Bootstrap 5.x with custom MD Anderson Cancer Center branding. The system emphasizes accessibility, usability, and professional healthcare aesthetics.

Technology Stack

Frontend Technologies

  • Bootstrap 5.x - CSS framework for responsive design
  • FontAwesome Pro 5.15.4 - Icon library
  • jQuery 3.7.0 - JavaScript library
  • SweetAlert2 - Enhanced modals and alerts
  • AOS (Animate On Scroll) - Animation library
  • FullCalendar 6.1.15 - Calendar component

Design Principles

  • Accessibility First - WCAG 2.1 AA compliance
  • Mobile Responsive - Mobile-first design approach
  • MD Anderson Branding - Consistent institutional identity
  • User Experience - Intuitive and efficient workflows
  • Progressive Enhancement - Graceful degradation

Usage Guidelines

Design Principles

Do's

  • Use consistent spacing (0.5rem, 1rem, 1.5rem, 2rem)
  • Follow MD Anderson brand colors
  • Ensure proper contrast ratios
  • Use semantic HTML elements
  • Test with keyboard navigation

Don'ts

  • Don't use arbitrary colors outside the palette
  • Don't rely solely on color to convey information
  • Don't use too many font weights or sizes
  • Don't create custom components without consulting this guide
  • Don't override Bootstrap defaults without good reason

Implementation Guidelines

CSS Organization

  1. Bootstrap CSS - Load first for base styles
  2. FontAwesome - Icon library
  3. Third-party libraries - AOS, SweetAlert2, DataTables
  4. Custom styles.css - System-specific overrides and components

JavaScript Libraries

  1. jQuery 3.7.0 - DOM manipulation and AJAX
  2. Bootstrap JS - Component interactions
  3. SweetAlert2 - Enhanced modals and alerts
  4. AOS - Scroll animations
  5. Custom scripts - Application-specific functionality
